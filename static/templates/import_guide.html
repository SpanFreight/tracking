Container Import Guide

Reference for importing containers using Excel or CSV files

Required Columns

These columns must be present in your import file:

Column Name Description
container_number The container identification number (e.g., ABCD1234567)
container_type Type of container (20GP, 40HC, etc.)

Optional Columns

These columns may be included but are not required:

Column Name Description
loading_port Port of loading
final_destination Final destination of container
opr OPR information
arrival_date Expected arrival date (YYYY-MM-DD)
bl_number Bill of Lading number
stripping_date Date when container is expected to be unloaded/stripped. This date is automatically set when a container is discharged from a vessel.
status Current status (loaded, discharged, emptied, full)
date Date of status (YYYY-MM-DD)
location Current container location
notes Additional notes

Bulk Import Containers

When performing bulk imports, please note the following required column information:

Required Column Information

  • container_number and container_type are mandatory fields
  • stripping_date is the date when a container is unloaded/stripped of its contents. This is automatically set to match the discharge date when a container is discharged from a vessel, or can be manually specified.
  • Tracking stripping dates helps with planning warehouse operations and monitoring container detention periods.
  • All dates should be in YYYY-MM-DD format for best results.

The system will validate your data during import and display error messages if any required fields are missing or incorrectly formatted.
